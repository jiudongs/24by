###### 七级大标题 ctrl+7

> Tips:最大就是7，不能在大了。 数字越大，文字越小
>
> 类似本段提示前面加竖线的语法： ”> “内容 箭头后+空格

### 有序列表语法：

1. 数字点空格，后输入列表内容
2. 自动生成，如果想取消 行号，连续两次回车。

### 无序列表

* 星号加空格，如需跳出无序列表，连续两次回车
